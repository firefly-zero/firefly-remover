# firefly-remover

System app for [Firefly Zero](https://fireflyzero.com/) for removing apps.

## Installation

1. [Install firefly-cli](https://github.com/firefly-zero/firefly-cli?tab=readme-ov-file#installation)
1. Run `firefly_cli import sys.remover`

After it is installed, select "remove" from the app menu in launcher and it will launch the remover.
